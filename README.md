# so_long
A 2d game made with the MLX library where the player must collect Bitcoins!
## Authors

- [@paaime](https://www.github.com/paaime)


## Installation

To run the program

```bash
  git clone https://github.com/paaime/so_long.git
  cd so_long
  make
  ./So_long [map file]
```
